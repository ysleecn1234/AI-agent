## 1. Auto 모드 (일반 업무 채팅)

### SIMPLE 난이도 (빠른 질문 — Router가 경량 모델 배정)

```
1. 안녕하세요
2. 오늘 날씨 어때?
3. HTTP 상태코드 404가 뭐야?
4. Python에서 리스트 정렬하는 방법 알려줘
5. JWT 토큰이 뭔지 간단히 설명해줘
6. Git에서 브랜치 만드는 명령어 알려줘
7. SQL에서 JOIN 종류 알려줘
8. REST API와 GraphQL 차이점 한 줄로
```

### COMPLEX 난이도 (심층 분석 — 다단계 파이프라인 풀가동)

```
9. 우리 회사에서 마이크로서비스 아키텍처로 전환하려고 하는데, 모놀리식 대비 장단점과 전환 전략을 단계별로 알려줘
10. Kubernetes vs Docker Swarm 비교해서 100명 규모 회사에 적합한 컨테이너 오케스트레이션 도구 추천해줘. 비용, 학습곡선, 운영 부담 포함해서
11. CI/CD 파이프라인을 GitHub Actions로 구축하려고 해. 테스트 자동화, 스테이징 배포, 프로덕션 배포까지 포함하는 워크플로우 YAML 예시 작성해줘
12. 우리 서비스가 일일 활성 사용자 10만 명인데, 데이터베이스 성능이 저하되고 있어. PostgreSQL 기준으로 쿼리 최적화, 인덱스 전략, 캐싱 레이어 도입 방안을 종합적으로 분석해줘
13. 2026년 현재 AI/ML 엔지니어 채용 시장 트렌드와 연봉 수준, 효과적인 채용 전략을 알려줘
14. 제로트러스트 보안 모델을 우리 회사에 적용하려면 어떤 단계로 진행해야 해? 네트워크, 인증, 데이터 각 레이어별로 설명해줘
15. AWS vs GCP vs Azure 클라우드 비용 비교. 월 서버 비용 500만원 규모의 SaaS 서비스 기준으로 각 클라우드별 예상 비용과 장단점 분석해줘
```

## 2. 프리미엄 모델 비교 테스트

> 아래 3개 프롬프트를 4개 프리미엄 모델로 각각 돌려서 비교

### 프롬프트 A: 코드 생성 (논리적 사고력 테스트)
```
Python으로 비동기 웹 크롤러를 만들어줘.
요구사항:
- aiohttp + asyncio 사용
- 동시 요청 수 제한 (세마포어)
- 에러 핸들링 및 재시도 로직
- 크롤링 결과를 JSON으로 저장
- 진행 상황 로깅
전체 코드와 함께 각 부분의 설계 의도를 설명해줘.
```

### 프롬프트 B: 분석/전략 (종합적 사고 테스트)
```
우리 회사(직원 100명, B2B SaaS)가 SOC 2 Type II 인증을 받으려고 해.
1. 준비부터 인증 획득까지 전체 로드맵 (타임라인 포함)
2. 각 단계별 필요한 인력과 예상 비용
3. 가장 많이 실패하는 항목과 대비 방법
4. 인증 유지를 위한 지속적 관리 체계
실제 사례 기반으로 현실적인 조언을 해줘.
```

### 프롬프트 C: 실시간 검색 (Perplexity 강점 테스트)
```
2026년 4월 현재 글로벌 AI 규제 동향을 정리해줘.
EU AI Act 시행 현황, 한국 AI 기본법 진행 상황, 미국의 AI 행정명령 이후 변화를 포함해서.
각 국가/지역별로 우리 같은 AI 서비스 회사가 준수해야 할 핵심 요구사항을 표로 정리해줘.
```
