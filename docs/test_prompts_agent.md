> ISOR 플랫폼 테스트용 — IT 회사 내부 에이전트

## 1. 에이전트 생성 테스트 (Step 1: 대화형 생성)

### 에이전트 A: 보안 컨설턴트
```
보안 관련 질문에 답변하는 에이전트를 만들고 싶어.
회사 내부 보안 정책, 개인정보보호법, 정보보안 가이드라인에 대해 직원들이 쉽게 물어볼 수 있게.
말투는 친절하지만 보안 사항은 단호하게 안내하는 스타일로.
```

### 에이전트 B: 코드 리뷰어
```
코드 리뷰를 도와주는 에이전트를 만들려고 해.
Python, TypeScript 위주로 코드 품질, 보안 취약점, 성능 개선점을 짚어주고,
개선된 코드 예시까지 제공해줬으면 좋겠어.
시니어 개발자처럼 건설적인 피드백을 주는 스타일로.
```

### 에이전트 C: 회의 정리봇
```
회의록을 정리해주는 에이전트가 필요해.
회의 내용을 넣으면 참석자, 안건, 결정사항, 액션아이템으로 구조화해서 정리해줘.
마크다운 표 형태로 깔끔하게 출력하고, 누락된 정보가 있으면 물어봐주는 스타일로.
```

### 에이전트 D: 온보딩 가이드
```
신규 입사자를 위한 온보딩 안내 에이전트를 만들어줘.
IT 장비 수령, 계정 생성, VPN 설정, 사내 시스템 접속 방법, 필수 교육 안내 등을 답변.
처음 입사한 사람도 이해하기 쉽게 친근하고 자세하게 설명하는 스타일.
```

## 2. 생성한 에이전트로 대화 테스트

### 보안 컨설턴트 테스트 대화
```
1. 외부 USB를 회사 노트북에 꽂아도 돼?
2. 재택근무할 때 공용 와이파이 써도 되나요?
3. 퇴직한 직원의 계정은 언제까지 유지해야 해?
4. 고객 데이터를 이메일로 외부에 전송해야 하는데 어떻게 해야 해?
5. 비밀번호 정책이 어떻게 되는지 알려줘
```

### 코드 리뷰어 테스트 대화
```
1. 이 코드 리뷰해줘:
def get_user(id):
    conn = psycopg2.connect("dbname=mydb user=admin password=1234")
    cur = conn.cursor()
    cur.execute(f"SELECT * FROM users WHERE id = {id}")
    return cur.fetchone()

2. 이 TypeScript 코드 개선점 알려줘:
async function fetchData() {
  const res = await fetch('/api/data')
  const data = await res.json()
  return data
}

3. Python에서 대용량 CSV 파일(10GB)을 메모리 효율적으로 처리하는 코드 작성해줘

4. 이 함수에서 보안 취약점 찾아줘:
def login(request):
    username = request.POST['username']
    password = request.POST['password']
    query = f"SELECT * FROM users WHERE username='{username}' AND password='{password}'"
    user = db.execute(query)
    if user:
        return redirect('/dashboard')
```

### 회의 정리봇 테스트 대화
```
1. 다음 회의 내용 정리해줘:
오늘 참석자는 김팀장, 이대리, 박사원이고,
신규 프로젝트 일정 논의했어. 김팀장이 4월 중으로 MVP 완성하자고 했고,
이대리가 백엔드 API는 2주면 된다고 했어. 박사원은 프론트 디자인 시안을
다음주 수요일까지 가져오기로 했어. 아 그리고 QA는 외부 업체 쓸지
내부에서 할지 다음 회의에서 결정하기로 했어.

2. 이번 회의에서 빠진 정보 있어?

3. 액션아이템만 따로 정리해서 담당자별로 보여줘
```

### 온보딩 가이드 테스트 대화
```
1. 안녕하세요, 오늘 첫 출근인데 뭐부터 해야 하나요?
2. 노트북은 어디서 받아요?
3. 사내 메신저는 뭐 쓰나요? 설치 방법 알려주세요
4. VPN 연결은 어떻게 해요?
5. 필수로 들어야 하는 교육이 있나요?
```

## 3. 에이전트 추천 기능 테스트

> AI Hub 메인 페이지에서 추천 기능이 작동하는지 확인

```
검색: "보안"  → 보안 컨설턴트 에이전트가 추천되는지
검색: "코드"  → 코드 리뷰어 에이전트가 추천되는지
검색: "회의"  → 회의 정리봇 에이전트가 추천되는지
검색: "입사"  → 온보딩 가이드 에이전트가 추천되는지
```
